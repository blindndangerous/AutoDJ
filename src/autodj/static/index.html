Settings
Mood
Bias the next-track picker toward a target BPM curve. E.g. wakeup ramps 70 → 130 BPM, chill stays at 75 BPM, slide arches. Built-ins plus any user presets in config.toml.
Audio treatment layered on every crossfade. echo out tail-throws the outgoing track with feedback delay; reverb tail bleeds a 4 s hall reverb past the cut; high-pass sweep introduces the new track muffled then opens up; low-pass sweep filters the outgoing into a build; tape stop slows the outgoing to a halt over 4 s; gate stutter chops the outgoing into rhythmic bursts (sample-accurate worklet); noise riser layers a synthesised white-noise build (bright); noise drop opposite of the riser — bright crash falling to silence; backspin reverses the outgoing tail with vinyl-friction decel (2.0×→0.05×); forward spin push-forward release (1.0×→3.0× accel); cross-EQ swap hands bass duties from the outgoing to the incoming track; bitcrusher degrades bit-depth + sample-rate (8-bit-console feel, AudioWorklet); flanger adds a swirly modulated delay; pitch swell sweeps pitch upward (preservesPitch off); telephone 500-2800 Hz band-pass + tanh saturation (sounds like a phone call); chorus three detuned voices for a thick double; submerge heavy lowpass + reverb wash (underwater); vinyl wow drunk-turntable pitch wobble (60 Hz polling, ±25 % depth); freeze captures the last 150 ms and loops with fade-out (granular, AudioWorklet); glitch random buffer slicing + reorder (ring-buffer worklet); scratch turntablist 4-pass back-and-forth over a 250 ms slice; beat repeat stamps a 250 ms slice 8 times across the tail (Pioneer Loop Roll); sidechain pump rhythmic 4-on-the-floor amplitude duck @ 120 BPM; reverse reverb reverse'd reverb tail that swells INTO the cut; air horn synth dub-siren with 220-880 Hz pitch sweep, layered with the music. random picks one per crossfade; rotate cycles through every effect in order.
DJ-mix toggles
Restrict next-track picks to harmonically related keys. compatible is the classic Camelot rule (same position, ±1, or relative major/minor) — broadest pool. strict only allows the exact same key. neighbour walks the wheel one step at a time, no mode flips. mood change swaps major↔minor only (energy stays, mood flips). energy boost picks +2 around the wheel — each track is two semitones up, so the set lifts over time. Pick off to disable the harmonic filter completely.
Pitch-stretch the incoming track up to ±8 % so its BPM matches the outgoing track during the crossfade — beats line up like a real DJ deck.
Snap the crossfade start to the nearest 8-bar boundary using the cached beat grid. Keeps transitions on musical phrase ends instead of mid-bar.
Auto-detect each track's outro start + intro end on first play. Crossfade lands between them so the new track's first downbeat replaces the old track's outro — no cold cuts into 4-bar intros.
Playback
Bass-cut the outgoing track during the overlap so its kicks don't muddy the incoming track's bass. The trick pro DJs use when manually mixing.
Each next pick is the most sonically distant track from whatever is currently playing — A → far(A) → far(far(A)) → … — so the set drifts through the library rather than clustering around the seed. Surprise mode, not a true random shuffle.
Uniformly random next pick — ignores similarity, key, BPM, energy, the lot. Bumps the recently-played window so the same track doesn't reappear too soon. Toggle off mid-set and the auto-DJ resumes similarity from the song that's currently playing.
Every next pick stays similar to the seed track (the song the session started from), not to whatever just played. Prevents the long sonic drift you get from chained similarity hops, while still letting Discovery break out periodically when enabled.
Display timed (LRC) or plain lyrics in the Lyrics card and the CLI panel. Toggle off if you'd rather not see them. Default: on.
Apply embedded ReplayGain track-gain tags so every track plays at a consistent loudness. Tracks without tags play unchanged. Targets −14 dB (Spotify-ish reference) by default.
Pick which sound card / headphones the browser plays through. Works in Chromium / Edge (full support) and Firefox 116+. Safari has not shipped setSinkId yet — it falls back to the system default. Browsers hide device names until you grant microphone permission once: click Show device names to do that (the permission is released immediately, no audio is captured). Selection saved per browser in localStorage; the server-side CLI player has its own --device flag and [playback] audio_device config.
Pick BPM and energy targets from the local time of day — gentler in the morning (6-10 am, 60-90 BPM), peak in the evening (6-10 pm, 115-140 BPM), chill late at night. Ignored when an explicit Preset is selected.
How long the gain ramp between tracks lasts. 2-3 s feels natural for most music; 5-8 s for ambient or testing transition effects. 0 = instant cut.
BPM range
Hard filter on track tempo. Picks only fall within this range; tracks with unknown BPM always pass. Leave blank to disable.
Discovery
Periodically inject a sonically distant track to break the auto-DJ out of a similarity loop. Picks at random from the bottom quartile of cosine similarity. Disable to stay strictly on-vibe.
